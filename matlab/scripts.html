Matlab Scripts

Scripts are files that execute a series of commands and can include variables, loops, and conditionals.

% Example 1: Simple script
x = 10;
y = x^2;
disp(y);

% Example 2: Script with conditional
if x > 5
disp('x is greater than 5');
end

% Example 3: Script with a for loop
for i = 1:5
disp(i);
end

% Example 4: Script to compute factorial
n = 5;
fact = 1;
for i = 1:n
fact = fact * i;
end
disp(fact);

% Example 5: Script to solve a quadratic equation
a = 1; b = -3; c = 2;
rootsOfEq = roots([a, b, c]);
disp(rootsOfEq);

% Example 6: Script using a function
result = squareNumber(4);
disp(result);

% Example 7: Script with a while loop
count = 0;
while count < 5
disp(count);
count = count + 1;
end

% Example 8: Script with a switch statement
grade = 'B';
switch grade
case 'A'
disp('Excellent');
case 'B'
disp('Good');
otherwise disp('Needs improvement');
end

% Example 9: Script to plot a sine wave
x = linspace(0, 2*pi, 100);
y = sin(x);
plot(x, y);
% Example 10: Script with user input
userInput = input('Enter a number: ');
disp(['You entered: ', num2str(userInput)]);
