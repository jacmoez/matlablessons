Matlab Flow Controls

MATLAB provides several flow control statements to control the execution of code. These include conditional statements (if, else, elseif, switch), loops (for, while), and jump statements (break, continue, return). Below is an overview of MATLAB flow control mechanisms:

if, else, elseif (Conditional Statements)
These statements allow for decision-making in code based on conditions.
x = 5;
if x > 10
disp('x is greater than 10');
elseif x == 5
disp('x is equal to 5');
else
disp('x is less than 10');
end

fruit = 'apple';
switch fruit
case 'apple'
disp('This is an apple.');
case 'orange'
disp('This is an orange.');
otherwise
disp('Unknown fruit.');
end

for i = 1:5
disp(['Iteration: ', num2str(i)]);
end

n = 0;
while n < 5 disp(['Current value of n: ', num2str(n)]);
n = n + 1;
end

for i = 1:10
if i == 5
break; % Exit the loop when i equals 5
end
disp([' i=', num2str(i)]);
end


for i = 1:5
if i == 3
continue; % Skip the iteration when i equals 3
end
disp([' i=', num2str(i)]);
end
