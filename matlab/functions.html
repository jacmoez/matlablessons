Matlab Functions

MATLAB functions are blocks of code that perform specific tasks. They are reusable and help structure and organize MATLAB code. MATLAB functions can be built-in or user-defined. Types of MATLAB Functions: Built-in Functions: Predefined functions provided by MATLAB, such as mathematical, statistical, or plotting functions. User-defined Functions: Custom functions created by users to perform specific tasks.

User-defined Functions

% Example 1: Basic function with one input and output
function y = squareNumber(x)
y = x^2;
end

% Example 2: Function with two inputs and one output
function result = multiply(a, b)
result = a * b;
end

% Example 3: Function with multiple outputs
function [sumResult, diffResult] = sumAndDiff(x, y)
sumResult = x + y;
diffResult = x - y;
end

% Example 4: Function to calculate area of a circle
function area = circleArea(radius)
area = pi * radius^2;
end


% Example 5: Function to calculate factorial
function result = factorial(n)
result = 1;
for i = 1:n
result = result * i;
end
end

% Example 6: Function with a variable number of inputs
function result = sumVarargin(varargin)
result = sum([varargin{:}]);
end

% Example 7: Function that calls another function
function output = callAnotherFunction(a)
output = squareNumber(a);
end

% Example 8: Function with no input arguments
function greet()
disp('Hello, world!');
end

% Example 9: Recursive function for factorial
function result = recursiveFactorial(n)
if n == 0
result = 1;
else
result = n * recursiveFactorial(n - 1);
end
end

% Example 10: Anonymous function
square = @(x) x^2;
disp(['Square of 4 using anonymous function is ', num2str(square(4))]);

% Anonymous function to add two numbers
addTwoNumbers = @(x, y) x + y;
disp(['Sum of 3 and 5 using anonymous function is ', num2str(addTwoNumbers(3, 5))]);

% Anonymous function to find the maximum of two numbers
maxValue = @(x, y) max(x, y);
disp(['Maximum of 7 and 4 using anonymous function is ', num2str(maxValue(7, 4))]);

% Anonymous function to compute the cube of a number
cube = @(x) x^3;
disp(['Cube of 3 using anonymous function is ', num2str(cube(3))]);

% Anonymous function to calculate the hypotenuse of a right triangle
hypotenuse = @(a, b) sqrt(a^2 + b^2);
disp(['Hypotenuse of a right triangle with sides 3 and 4 is ', num2str(hypotenuse(3, 4))]);

% Anonymous function to compute the area of a rectangle
rectangleArea = @(length, width) length * width;
disp(['Area of a rectangle with length 5 and width 3 is ', num2str(rectangleArea(5, 3))]);

% Anonymous function to check if a number is even
isEven = @(x) mod(x, 2) == 0;
disp(['Is 8 even? ', num2str(isEven(8))]);

% Anonymous function to calculate compound interest
compoundInterest = @(P, r, t, n) P * (1 + r/n)^(n*t);
disp(['Compound interest after 5 years is ', num2str(compoundInterest(1000, 0.05, 5, 12))]);

% Anonymous function to calculate the absolute difference
absDifference = @(x, y) abs(x - y);
disp(['Absolute difference between 7 and 10 is ', num2str(absDifference(7, 10))]);

% Anonymous function to compute the sine of a number
sineFunction = @(x) sin(x);
disp(['Sine of pi/4 is ', num2str(sineFunction(pi/4))]);

Some Build-in Funtions

size(A): Returns the dimensions of matrix A.
length(A): Returns the length of the largest dimension of array A.
inv(A): Computes the inverse of matrix A.
det(A): Computes the determinant of matrix A.
eye(n): Creates an identity matrix of size n x n.
zeros(n,m): Creates an n x m matrix of zeros.
ones(n,m): Creates an n x m matrix of ones.
transpose(A): Computes the transpose of matrix A (same as A').
