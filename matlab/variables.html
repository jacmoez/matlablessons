Matlab Variables

A variable is a named storage that holds data. Variables allow programs to store, retrieve, and manipulate data values. They are essential building blocks in software development because they enable dynamic data handling.

% Example 1: Scalar variable
x = 5;

% Example 2: String variable
name = 'Car Automation';

% Example 3: Boolean variable
isAutonomous = true;

% Example 4: Matrix variable
M = [1, 2, 3; 4, 5, 6; 7, 8, 9];

% Example 5: Struct variable
car.speed = 100;
car.model = 'Sedan';

% Example 6: Cell variable
C = {5, 'Vehicle', [1, 2, 3]};

% Example 7: Double precision variable
distance = 150.75;

% Example 8: Logical variable
sensorActive = false;

% Example 9: Time variable using `datetime`
currentTime = datetime('now');

% Example 10: Creating a complex number
z = 4 + 3i;
